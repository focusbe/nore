前端页面可视化开发工具

技术选型：vue electron webpack typescript

## 主要功能：
1. 可视化添加，移动，编辑组件
2. 组件包含：图片、文字、容器等

## 使用方法：

```
npm install
npm run dev #启动开发模式
npm run build #运行打包构建
```
